# task2-phishing


# Objective
To analyze a suspicious email and identify phishing indicators.

---

# Sample Email Overview

Sender: support@paypa1-security.com  
Subject: Urgent: Account Suspension Notice  

---

# Phishing Indicators Identified

1. Spoofed Domain Name
   - "paypa1" instead of "paypal"

2. Suspicious URL
   - Link redirects to fake domain

3. Urgent Language
   - "Act within 24 hours"
   - "Account will be suspended"

4. Generic Greeting
   - "Dear Customer"

5. SPF/DKIM Failure (Header Analysis)

6. Threat-based Social Engineering
